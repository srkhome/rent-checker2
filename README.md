## 範例 📝

- 行政區：北屯區  
- 是否為套房：yes  
- 屋齡：6  
- 坪數：20.5  
- 現在租金：18,000  

👉 系統會判斷該區間的租金範圍，並輸出：  
- O（綠色），表示符合規範  
- 顯示：對應分段、租金區間、現在租金（不會顯示最高租金，因為符合）

---

## 修改基礎數據 🔧
若需更新基準值（行政區、戶數、調整率…），請編輯 `index.html` 中的以下常數：
- `HOUSEHOLD` → 各行政區非套房基準
- `STUDIO` → 各行政區套房基準
- `ADJUST_RATES` 與 `BRACKETS` → 屋齡調整率與分段設定
